## ℹ️ What is api-manager?

api-manager is a simple tool to help you manage API accounts in one place. It works well with the "All API Hub" plugin. You can quickly import your accounts from that plugin with one click. This tool makes working with different API accounts easier by organizing them for you.

This project is designed for Windows users who want a clear way to handle API accounts without complex setup or extra software. You do not need to be a programmer to use it.

---

## 🖥️ System Requirements

To run api-manager, you will need:

- A Windows PC running Windows 10 or later  
- At least 4 GB of RAM (8 GB recommended for better performance)

- Internet connection for downloading and updating

## 🛠️ How to Use api-manager  

Once installed, open api-manager by clicking its icon. The interface is designed to be clear and easy:

- **Import Accounts**  
  Use the import feature to bring in your accounts from the "All API Hub" plugin. Just click the "Import from All API Hub" button. This saves you time by avoiding manual entry.  
- **Manage Accounts**  
  You can view all your API accounts in one list. Select an account to edit details or delete it if you no longer need it.  
- **Add New Accounts**  
  If you have other API accounts, add them directly in the app by clicking "Add Account" and entering the details.  
- **Quick Access**  
  Use the tool to quickly copy account information when needed for your workflow or development.  

The app keeps your information safe and easy to reach. You do not need to open multiple windows or remember different account details.

## ❓ Troubleshooting

If you run into problems:

- Make sure your Windows is updated to the latest version.  
- Check that the downloaded file is complete and not corrupted. If unsure, download again from the release page.  
- If you cannot open the setup file, right-click it and select "Run as administrator".  
- Disable antivirus temporarily if it stops the program from installing or running.  
- For issues with importing accounts, verify the "All API Hub" plugin is installed correctly and you have exported or saved your account data.

You can open the tool’s "Help" section for common tips or visit the repository’s issues page on GitHub to see if others face similar problems.
